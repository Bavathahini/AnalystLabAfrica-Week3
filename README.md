# AnalystLab Africa – Week 3: Statistics & Probability

## Overview
This repository contains my Week 3 internship task for the AnalystLab Africa Data Science Internship Program.

The objective of this task was to perform statistical analysis on two cleaned datasets by applying descriptive statistics, probability concepts, hypothesis testing, and correlation analysis using Python.

---

## Datasets Used

1. Titanic Cleaned Dataset
2. Housing Cleaned Dataset

---

## Objectives

- Perform descriptive statistical analysis
- Calculate mean, median, variance and standard deviation
- Explore probability distributions
- Conduct hypothesis testing
- Analyze correlation between variables
- Visualize statistical findings using charts

---

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
